# smartmet-library-regression

Part of [SmartMet Server](https://github.com/fmidev/smartmet-library-spine). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-library-spine) for an overview of the ecosystem.

## Overview

The regression library provides a regression testing framework used across SmartMet Server tools and libraries. It enables automated comparison of program outputs against expected reference results.

## Features

- Reference output comparison for SmartMet command-line tools
- Test case management and result reporting
- Used in CI/CD pipelines for SmartMet components

## Usage

Used by the test suites of [smartmet-qdtools](https://github.com/fmidev/smartmet-qdtools), [smartmet-qdcontour](https://github.com/fmidev/smartmet-qdcontour), and other SmartMet components.
